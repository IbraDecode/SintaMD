# SintaMD

Bot WhatsApp multi-device berbasis Node.js dengan fitur lengkap untuk otomasi dan hiburan.

## Deskripsi

SintaMD adalah bot WhatsApp yang mendukung multi-device, dibangun menggunakan Baileys. Bot ini menyediakan berbagai fitur seperti AI, downloader media, manajemen grup, dan banyak lagi.

## Fitur Utama

- **AI & Otomasi**: Integrasi dengan Gemini, OpenAI, dan fitur AI lainnya.
- **Downloader**: Mendukung TikTok, YouTube, Instagram, Facebook, dan platform lainnya.
- **Grup Management**: Welcome/leave messages, anti-link, promote/demote, dll.
- **Stiker & Media**: Pembuatan stiker, konversi media, OCR.
- **Informasi & Tools**: Cek IP, speedtest, translate, TTS, dll.
- **Premium & Limit**: Sistem pengguna dengan limit harian dan fitur premium.

## Instalasi

1. Clone repository ini:
   ```
   git clone https://github.com/IbraDecode/SintaMD.git
   cd SintaMD
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Jalankan bot:
   ```
   npm start
   ```
   Atau untuk development:
   ```
   npm run dev
   ```

## Penggunaan

- Pairing code akan ditampilkan saat pertama kali menjalankan.
- Gunakan perintah dengan prefix "." (contoh: .menu).
- Lihat menu lengkap dengan perintah .menu.
